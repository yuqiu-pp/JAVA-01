###### 1、（选做）总结一下，单例的各种写法，比较它们的优劣。
```
singleton
- 饿汉模式 ：启动时实例化对象，影响应用启动时间；
- 懒汉模式 ：使用时才实例化对象(延迟加载)，但每次都会获取锁，并发性能低；
- 双重检查锁模式 ：延迟加载，并发性能高；
- 枚举模式 ：可以防止反射；
- 静态内部类	：延迟加载，并发性能高。实现简单；
- 线程之间的单例，类似于ThreadLocal

```
