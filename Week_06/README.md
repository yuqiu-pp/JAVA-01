## 2.22日作业
###### 2.（必做）基于电商交易场景（用户、商品、订单），设计一套简单的表结构。
```
1.用户模型
  ID
  手机号
  姓名
  密码（用户ID、密码）
  平台账号信息（用户ID、平台余额）
  平台交易记录（用户ID、交易金额、交易时间）
  地址列表（用户ID、省、市、区、门牌号）
  购物车商品列表（用户ID、商品ID）
  历史订单列表（用户ID、订单ID）

2.商品模型
  ID
  名称
  详情（商品ID、商品介绍）
  评价分数
  评论（商品ID、用户ID、评价）

单品模型 -- SKU相关属性
  ID
  商品颜色
  商品规格
  商品图片
  商品状态（上架、下架）
  库存（单品ID、数量）
  单价（单品ID、价格）
  促销（单品ID、价格、有效时间、数量）
  


3.订单模型
  ID
  商品ID
  商品价格
  购买数量
  商品快照
  订单状态：创建成功，未支付、已支付、待发货、已发货、已送达、退货申请提交、退货进行中、退货完成、订单结束
  
  创建时间
  修改时间
  总金额
  购买人信息（用户ID）
  邮寄信息（用户ID、姓名、省、市、区、门牌、邮编）
  快递信息（订单ID、快递公司、轨迹、状态）

```
